# VIVC Pedigree Explorer

Interactive grape variety pedigree and marker visualization tool built on the [Vitis International Variety Catalogue](https://www.vivc.de).

**Live app:** [vivc-pedigree-explorer.streamlit.app](https://vivc-pedigree-explorer.streamlit.app)

## Features
- 🔍 Pedigree network visualization (ancestors & descendants)
- 🌿 Descendants search with BFS traversal
- 📅 Timeline chart by year of crossing
- 🌳 Founder composition analysis
- 🪴 Rootstock data
- 📋 Full searchable data table
- 📊 Data quality summary

## Run locally
```bash
pip install -r requirements.txt
streamlit run app.py
```
